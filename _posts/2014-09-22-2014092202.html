Python 错误处理【'ascii' codec can't decode byte】

（ruilovetv 版权所有，转载请附出处）

出错提示，如下：


'ascii' codec can't decode byte 0xe4 in position 0: ordinal not in range(128)

处理方法1：在代码首行增加如下代码


import sys  
reload(sys)  
sys.setdefaultencoding('utf8')

缺陷：导致无法显示print结果。

处理方法2：在代码中找出转码出错的变量，添加.encode('utf-8')方法，并对在字符串前加u


text = u"测试" + text
file_object = open(file_name, 'w')
file_object.write(text.encode('utf-8'))

综合比较，推荐处理方法2。
